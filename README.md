# GrabAPass
Cloud-Based Event Ticketing System
Project Overview

This project is a cloud-based event ticketing platform that allows organizers to create events, manage seating layouts, sell tickets online, and validate entry using QR codes. The system supports real-time ticket sales tracking and is designed to handle high-demand ticket releases (flash sales).

The platform follows a modern Angular + Rust full-stack architecture with PostgreSQL as the relational database and Docker for containerized deployment.
